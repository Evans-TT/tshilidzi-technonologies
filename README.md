# Tshilidzi Technologies Static Site

React + Vite rebuild of the Wix site, prepared for static deployment.

## Run Locally

```bash
npm install
npm run dev
```

## Build For GitHub Pages

```bash
npm run build
```

The deployable static site is generated in `dist/`.

## Search Indexing

After deploying, submit this sitemap in Google Search Console:

```text
https://www.tshilidzitechnologies.co.za/sitemap.xml
```

The site also ships with `robots.txt`, canonical metadata, Open Graph metadata, Twitter metadata, and Schema.org structured data.
